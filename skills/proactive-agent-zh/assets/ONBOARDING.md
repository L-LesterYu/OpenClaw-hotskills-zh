## 如何工作

当您的 Agent 看到此文件且 `state: not_started` 或 `in_progress` 时，它知道要帮助您完成设置。您可以：

1. **交互模式** — 在一次会话中回答问题（约 10 分钟）
2. **滴灌模式** — Agent 在几天内自然地问 1-2 个问题
3. **暂时跳过** — Agent 立即工作，从对话中学习

说"让我们开始入门"开始，或"稍后问我"滴灌。

## 核心问题

回答这些以帮助您的 Agent 了解您。留空跳过。

### 1. 身份
**我该怎么称呼您？**
> 

**您的时区是什么？**
> 

### 2. 沟通
**您希望我如何沟通？（直接/详细/简洁/随意）**
> 

**有什么我应该避免的讨厌的事情吗？**
> 

### 3. 目标
**您现在的主要目标是什么？（1-3 句话）**
> 

**对您来说 1 年后"获胜"是什么样的？**
> 

**当您成功时，理想的生活看起来/感觉是什么样的？**
> 

### 4. 工作风格
**您什么时候最有生产力？（早上/下午/晚上）**
> 

**您偏好异步沟通还是实时？**
> 

### 5. 上下文
**您目前在做什么？（项目、工作等）**
> 

**我应该了解的您工作/生活中的关键人物是谁？**
> 

### 6. Agent 偏好
**您的 Agent 应该有什么样的个性？**
>

## 入门后

一旦完成（或收集了足够的答案），Agent 将：
1. 用您的上下文更新 USER.md
2. 用个性偏好更新 SOUL.md
3. 将状态设为 `complete`
4. 启动主动模式

您可以随时通过编辑此文件或告诉您的 Agent 来更新答案。
